## Development

Built with:
- React
- TypeScript
- Tailwind CSS
- Vite

### Local Setup 
```bash
# Clone the repository
git clone https://github.com/Plasma-Blue/Kana

# Navigate to project directory
cd Kana

# Install dependencies
npm install

# Start development server
npm run dev
```
